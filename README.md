# Проведённая работа
В контексте данного проекта были реализованы:
* Прототип 3D игры с управлением транспортом с **hot-sit разделением экрана** расчитанным на 2 игрока
* Прототип **side-view** 3D игры (путь: ./Challenge 1/ ) с управлением самолётом в вертикальной плоскости для преодоления преград (по типу 3D версии flappy bird)
Помимо прочих уже известных рабочих процессов, во время разработки были затронуты темы:
* Реализация плавного управления транспортом c помощью функций FixedUpdate (), LateUpdate (), Awake ()
* Математика с кватернионами
* Разделение экрана с помощью настройки 2ух камер в иерархии
